<h1>SpringBoot를 활용한 쇼핑몰 프로젝트(클론코딩)</h1>

<h2>주요기능</h2>

1. 로그인, 회원가입
2. 쇼핑몰 물품 리스트 나열
3. 쇼핑몰 리스트 관리(관리자)
4. 장바구니 등록 및 구매(구매자)
